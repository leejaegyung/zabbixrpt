# 04. 검증 — 회귀 체크리스트 (Verify)

> 목표: 신규 Vue+Laravel 앱이 기존 `zabbix8.0.html`과 **사용자 식별 가능한 차이 0건**임을 증명.
> 기준 입력: 기존 HTML 스크린샷 + 동일 데이터(가상 Mock + 실 Zabbix).

## A. 기능 동등성

### 데이터 조회
- [ ] 전체 대시보드 갱신 → 호스트/문제/아이템 동시 로드, 전체 기본 선택
- [ ] 호스트/문제/아이템 개별 조회 동작
- [ ] 기간(시작/종료 date+time) → epoch 변환, 미지정 시 problem `recent:true`
- [ ] item: value_type 0/3, monitored, limit 300 / history 1000 유지

### 인프라 7대 스마트 분석
- [ ] 가동률(SLA) 계산값 = 기존
- [ ] 요주의 서버(알람 최다 Top) 동일
- [ ] 스토리지 포화 예측(선형회귀 D-day) 동일
- [ ] CPU Top5 / 메모리 Top5 평균값·정렬 동일
- [ ] 사용량 급증 감지(%p ≥10 / % ≥30 임계) 동일
- [ ] 네트워크 Top5(1Kbps 초과) 동일
- [ ] 카드 개별 토글, 전부 off 시 섹션 미렌더

### 필터 / 페이지네이션
- [ ] 호스트 상태/그룹 필터
- [ ] 문제 심각도(다중)/호스트/그룹 필터
- [ ] 아이템 호스트 필터
- [ ] 테이블 10건/페이지, 그리드 small 8 / medium 6 / large 3
- [ ] 데이터·필터 변경 시 페이지 1 리셋

### 설정 / 프로필
- [ ] 로고 업로드(base64) / 워터마크 / 그래프 크기 — draft→저장 반영, localStorage 영속
- [ ] 프로필 JSON 내보내기/불러오기 (토큰 제외 정책)
- [ ] 가상 데이터 로드
- [ ] 화면 표시 설정 모달 (섹션·분석 카드·PDF 포함)

### UX / 에러
- [ ] 커스텀 alert/confirm 모달, 토큰 발급 도움말 모달
- [ ] ErrorBoundary 동작 (렌더 오류 시 복구 화면)
- [ ] 네트워크/CORS/404/401/인증만료 한국어 문구 동등

## B. 디자인 동등성
- [ ] 헤더/패널/테이블/카드 레이아웃 픽셀 비교 (육안 차이 0)
- [ ] Tailwind 색상·간격·폰트·아이콘 동일
- [ ] 반응형(md/xl 그리드) 동일
- [ ] 한국어 문구 1:1 일치

- [ ] 데이터 0건·로딩 중 버튼 비활성
- [ ] 기존 HTML PDF와 레이아웃 비교

- [ ] Zabbix 7.0 (Bearer) 인증 성공

## E. 폐쇄망 / Docker
- [x] 멀티스테이지 이미지 빌드 성공 (app 174MB / web 94MB) — 의존성 전부 동봉
- [x] `docker compose up` 단일 명령 기동 (web:8090 → app php-fpm)
- [x] 런타임 외부 CDN/패키지 요청 0 — index.html·메인번들 외부 CDN 참조 0, PDF 라이브러리 로컬 청크 200
- [x] nginx SPA fallback + `/api`·`/up` 프록시 동작 (E2E HTTP 검증)
- [x] 환경변수(ZABBIX_API_URL/TOKEN/ALLOW_REQUEST_URL) compose 주입 동작
- [ ] 완전 air-gapped 빌드(베이스 이미지까지 사전 적재) — 운영 레지스트리 미러 환경에서 별도 확인 권장
